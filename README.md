# aidraw
## 介绍
使用stable-diffusion的web-api进行绘图的插件  
适配真寻，初次使用请将aidraw_setting.json文件放在zhenxun_bot/resources/text目录下  
## 使用：
文字成图  
`ait2i+描述(英文)`  
图片成图  
`aii2i+描述(英文)+图片`  
`aii2i+@群友/qq号/回复图片+描述(英文)`  
当前设置  
`drawshow+s/m 查看当前(设置)/(拥有的风格、尺寸调整方式)`  
修改设置  
`drawset+set_name+set_value`  
*加号组成部分以空格隔开*
